# Markdown

```

Negrito: (__EX__ ou **EX**);

Etalico: (_EX_ *EX*);

Negrito e Etalico: (__*EX*__)

Risco no meio da palavra: (~~EX~~);

Titulos: (#EX, ##,EX, ###EX);

Linha divisoria: (--- ou ***);

Lista numerada: não importa o que vai ter antes do ponto vai ir sempre na ordem EX: 1. 2. 3. ...(1. EX [Lina de baixo] 2. EX), e tambem tem como faser um sub item po exemplo do primeiro [1.EX] e so da tres espassos;

Lista não ordenada: para ficar com a bolinha toda preta: (*EX), para deixar a bolinha com o contorno preto e vazia por dentro: ( *EX) usa um espaco antes;

Lista de traefa: (- []EX) tarefa concluida (- [x]EX);

Emoji: (:Emoji);

Imagem: arastando a imagem ira apareser (![DESCRIÇÂO](https://...)), Fomatos recomendados: .png .jpg, Tamanho ta imagem recomendo não usar mais do que 500px de largura;

Links: ([DECRIÇÂO](https://...));

Tabelas: primeiro tem que colocar os titulos (Num | Nome | Nota);
Divisão: 				     (---|---|---);
Dados da tabela:			     (1 | Gustavo | 8,5);
					     (2 | José | 10,0);

Pults: (>EX)

Comandos: (`console.log(Holla Mundo);`) entre crazes, se quiser comando mais longo: ``` javascirpt
										   function holla() {
										      console.log(Holla Mundo
										   }
										   console.log(holla()
										   ```
